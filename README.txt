1- Ir a la ruta del package.json y generar el comando npm install por consola
2- Ejecutar la app con el comando : node app.js (desde la misma ruta del paso 1)
3- Entrar a http://localhost:5600

ACLARACION:
-Se necesita conexión a internet ya que la BD de mongo esta subida a https://mlab.com/ (tiene la misma estructura del json que me pasaron)
-Solo funciona el filtro por nombre (distingue entre mayusculas y minusculas) y el filtro por estrellas falto desarrollo.
Era mas fasil que  filtrara por el ng-repeat | filter. Pero la consigna pedia que hiciera una petición.
-En app.js  (donde se escuentra el server local) esta la alta y actualzación de hoteles (solo a nivel api).


Las tecnología usadas para la primera parte fueron nodejs, express, mongo y mongoose.
Las tecnología usadas para la segunda parte fueron bower, angular1.7, css, html, js y Bootstrap.
